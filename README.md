## 📂 Proyectos

🚖 Movilidad urbana y productividad económica en ciudades de Latinoamérica

🌎 Contexto del proyecto
El crecimiento urbano en LATAM ha generado problemas de congestión que afectan tanto la calidad de vida como la productividad económica. Este proyecto analiza la relación entre movilidad urbana y desempeño económico en distintas ciudades.

🔍 Análisis
Se realizó limpieza y transformación de datos para estandarizar métricas entre ciudades. Posteriormente, se aplicó análisis exploratorio y visualización para identificar patrones entre congestión, tiempos de traslado y productividad.

📈 Conclusiones
🚗 La alta congestión no garantiza mayor productividad
🏙️ Algunas ciudades logran mayor eficiencia con menor tráfico
📊 La relación entre movilidad y productividad no es directa
🌐 Existen diferencias importantes entre regiones

💡 Los resultados pueden apoyar decisiones en movilidad, planeación urbana y optimización de recursos.
🛠️ Skills: Python | Pandas | MatPlotlib

 📊 Análisis de embudo y retención para MercadoLibre

 🌎 Contexto del proyecto
En plataformas de e-commerce como Mercado Libre, entender el comportamiento de los usuarios es clave para mejorar la conversión y retención. Este proyecto analiza el recorrido del usuario desde su primera visita hasta la compra.

🔍 Análisis
Se construyó un embudo de conversión a partir de eventos clave (first_visit, select_item, add_to_cart, purchase).
Se aplicaron consultas en SQL para medir tasas de conversión entre etapas y analizar patrones de retención de usuarios.

📈 Conclusiones 
🛑 Existe una caída significativa en etapas intermedias del embudo
🛒 No todos los usuarios que exploran productos avanzan a compra
🔁 La retención de usuarios es un factor clave para mejorar conversión
📊 Optimizar etapas críticas puede generar alto impacto en ingresos

💡 El análisis permite identificar puntos de fricción en el embudo y oportunidades para mejorar la experiencia del usuario y la conversión.
